# Social Network web application
_Dwitter, a small social network web application ._

# Built with:
------------------
- Django
- HTML
- CSS (Bulma CSS framework)
- Sqlite3

# Features:
---------------------------------
User can:
- Create new account.
- Reset password.
- Change password.
- Login.
- Post short text-based dweet.
- Follow/ Unfollow other user profiles.
